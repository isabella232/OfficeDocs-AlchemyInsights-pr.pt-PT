# <a name="activate-visio"></a>Ativar Visio

Antes de um utilizador poder instalar e ativar a aplicação de ambiente de trabalho Visio ambiente de trabalho incluída no respetivo Visio Plano Online 2, tem de lhe ter sido atribuída uma licença Visio ambiente de [trabalho.](https://docs.microsoft.com/microsoft-365/admin/add-users/add-users)

**Nota** Visio Plano Online 1 não inclui uma versão de ambiente de trabalho do Visio.

Para ativar Visio:

1. Instale Visio a [partir do portal](https://portal.office.com/account#installs) A minha conta, se ainda não estiver instalado. Para obter ajuda, consulte [Instalar o Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1800).
2. Quando a instalação estiver completa, abra a aplicação do Visio. O Visio será provavelmente ativado de forma automática, pelo que pode começar a usá-lo de imediato. Se não for esse o caso, poderá ser-lhe pedido para iniciar sessão. Certifique-se de que inicia sessão com a mesma conta escolar ou profissional à qual a licença do Visio está associada. Siga as instruções para continuar a ativação. 

Para problemas de ativação, execute a [Remoção de](https://aka.ms/SARA-OfficeActivation-Alchemy) problemas de ativação no computador afetado.
