# <a name="buy-subscriptions-through-the-windows-365-products-site"></a>Comprar subscrições através do site Windows 365 produtos

Se ainda não tiver uma subscrição do Microsoft 365, pode comprar as suas subscrições do Windows 365 Empresas no site de produtos [do Windows 365](https://www.microsoft.com/windows-365/business/compare-plans-pricing?rtc=1). Utilize os passos seguintes para comprar uma subscrição do Windows 365 Empresas através da página de Windows 365 produtos.

1. Na página [Windows 365 Empresas,](https://www.microsoft.com/windows-365/business?rtc=1) **selecione Ver planos e preços**.
2. Na página seguinte, selecione a subscrição que pretende comprar e, em seguida, **selecione Comprar agora.**
3. Na página **Obrigado por escolher o Windows 365 Empresas,** siga os passos para configurar a sua conta.
4. No **passo 5 - Detalhes** de confirmação, se estiver pronto para  atribuir licenças aos utilizadores, selecione Começar para aceder à home page do Windows 365 em [https://windows365.microsoft.com](https://windows365.microsoft.com) .
5. Na home page Windows 365, na secção Ações **rápidas,** selecione **Gerir a sua organização**. Esta ação leva-o centro de administração do Microsoft 365 onde pode atribuir licenças a utilizadores.

Os utilizadores sem as funções de Administrador Global ou de Faturação podem utilizar a compra self-service para comprar uma subscrição do Windows 365 Empresas diretamente a partir do site de produtos do [Windows 365](https://www.microsoft.com/windows-365/business?rtc=1)Empresas. Um utilizador que compre uma subscrição desta forma tem uma vista limitada do [centro de administração do Microsoft 365,](https://go.microsoft.com/fwlink/p/?linkid=2024339)onde pode atribuir licenças para essa subscrição a outros utilizadores na respetiva organização. Atribuir uma licença a outro utilizador cria automaticamente um PC na Nuvem do Windows 365 Empresas para o mesmo, ao qual pode aceder a partir da home page do [Windows 365.](https://windows365.microsoft.com/)

> [!NOTE]
> A compra self-service não está disponível na Índia nem para clientes governamentais ou educacional.

Para saber mais sobre compras self-service, consulte as [FAQ sobre compras independentes.](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/self-service-purchase-faq)
