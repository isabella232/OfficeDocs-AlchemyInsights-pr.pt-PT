# <a name="error-code-30125-4-or-30125-1011-when-installing-office"></a>Código de erro 30125-4 ou 30125-1011 ao instalar Office

Poderá ver este erro se o seu software antivírus, firewall, definições de proxy ou ligação o impedirem de instalar Office. Eis algumas sms que pode experimentar para resolver o problema:
  
- Reinicie o computador e reinstale o Office

- Instalar Office com o [instalador Office offline](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA)

- Deslige temporariamente as definições de proxy

- Deslige temporariamente o software antivírus

- Desligar temporariamente a firewall

Para obter mais informações, consulte Código de [erro 30125-4 ou 30125-1011 ao instalar Office](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612?wt.mc_id=Alchemy_ClientDIA)
