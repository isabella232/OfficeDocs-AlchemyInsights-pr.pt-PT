# <a name="switch-subscriptions-to-increase-mailbox-size"></a>Mudar de subscrições para aumentar o tamanho da caixa de correio

Se o tamanho da sua caixa de correio já for de 100 GB, não existe forma de o aumentar.
  
Se o tamanho da caixa de correio for de 50 GB, pode aumente para 100 GB ao mudar para o [Office 365 Enterprise E3](https://products.office.com/business/office-365-enterprise-e3-business-software) [ou Office 365 Enterprise E5.](https://products.office.com/business/office-365-enterprise-e5-business-software) Eis os passos para mudar para uma subscrição diferente:
  
1. Na página centro de administração do Microsoft 365, vá para a página **Faturação** \> **[dos seus](https://go.microsoft.com/fwlink/p/?linkid=842054)** produtos.

2. Selecione a subscrição que pretende atualizar.

3. Na página de detalhes da subscrição, **selecione Atualizar**.

4. Localize o plano para o qual pretende atualizar e, em seguida, **selecione o botão** Atualizar.

5. Introduza o número de licenças de que precisa, escolha se quer pagar mensal ou anualmente e, em seguida, **selecione Finalidade da Compra.**

Para obter mais detalhes, [consulte Atualizar para um plano diferente.](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/upgrade-to-different-plan)

Se tiver uma Subscrição que suporte um arquivo no local, também pode obter armazenamento de [e-mail](https://docs.microsoft.com/office365/servicedescriptions/exchange-online-archiving-service-description/exchange-online-archiving-service-description)adicional ao ativar uma caixa de correio de arquivo. Uma caixa de correio de arquivo fornece-lhe armazenamento adicional e permite-lhe mover, copiar e procurar mensagens na caixa de correio principal e de arquivo. Para ver os passos sobre como ativar uma caixa de correio de arquivo, consulte [Ativar uma caixa de correio de Arquivo.](https://docs.microsoft.com/microsoft-365/compliance/enable-archive-mailboxes)
