# <a name="redirected-to-delve-after-you-click-onedrive"></a>Redirecionado para o Delve após clicar em OneDrive

Consulte o nosso Guia [de Remoção de Problemas detalhado.](https://docs.microsoft.com/sharepoint/support/sites/troubleshooting-guide-for-sites-stopped-at-provisioning)

Para resolver este problema, o administrador tem de conceder aos utilizadores o direito de criarem o respetivos sites Os Meus Sites. Isto deve-se ao facto OneDrive para Empresas página de Criação em Os Meus Sites.

Para atribuir este direito, siga estes passos:

1. No centro de administração do SharePoint, clique em **perfis de utilizador.**

2. Na secção **Pessoas,** clique em **Gerir Permissões de Utilizador.**

3. Adicione utilizadores que necessitem de permissões para criar o respetivos sites Os Meus Sites. Por predefinição, esta definição está definida para **Todos exceto os utilizadores externos**.

4. Após adicionar o utilizador, os utilizadores ou o grupo, certifique-se de que o utilizador, utilizador ou grupo adicionado está selecionado, desloco até à secção de permissões e, em seguida, selecione a caixa de verificação junto a Criar Site Pessoal (necessário para armazenamento **pessoal, newsfeed** e conteúdos **seguidos).**

5. Clique **em OK** e, em seguida, faça com que o utilizador navegue até à OneDrive de pesquisa para criar o site.
