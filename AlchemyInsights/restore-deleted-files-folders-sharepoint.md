# <a name="restore-deleted-files-or-folders-in-sharepoint"></a>Restaurar ficheiros ou pastas eliminadas no SharePoint

Se eliminar acidentalmente um ficheiro ou pasta no SharePoint, poderá recuperá-lo mais tarde a partir da reciclagem.

* [Restaurar ficheiros ou pastas eliminadas no SharePoint](https://support.microsoft.com/office/restore-items-in-the-recycle-bin-that-were-deleted-from-sharepoint-or-teams-6df466b6-55f2-4898-8d6e-c0dff851a0be)

Se muitos dos seus ficheiros do SharePoint foram eliminados, sobrescritados, danificados ou infetados por software mal formatado, pode restaurar a sua biblioteca do SharePoint inteira para uma data anterior.

* [Restaurar uma biblioteca de documentos](https://support.office.com/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a)
