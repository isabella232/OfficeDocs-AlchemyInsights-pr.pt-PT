# <a name="give-users-access-to-sharepoint-site"></a>Dar aos utilizadores acesso ao site do SharePoint

A funcionalidade de pedido de acesso permite às pessoas pedir acesso a conteúdo que, de momento, não têm permissão para ver. Enquanto proprietário do site, pode configurar a funcionalidade para lhe enviar um e-mail quando alguém pedir acesso a um site. Para obter mais informações, aceda [a Configurar e gerir pedidos de acesso.](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3)

Eis algumas ligações comuns sobre pedidos de acesso:

- [Access Denied" to Access Requests list or "Request approval failed" when you process a pending request in SharePoint Online](https://docs.microsoft.com/sharepoint/support/sharing-and-permissions/request-approval-failed)

- [Alterar definições de pedidos de acesso](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow)

- [Aprovar ou recusar pedidos de acesso a um site](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558)

- [Gerir convites para utilizadores externos (apenas no SharePoint Online)](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260)

- [Reenviar um convite](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560)
