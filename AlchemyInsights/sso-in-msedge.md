# <a name="single-sign-on-sso-in-microsoft-edge"></a>SSO (Single-sign on) no Microsoft Edge

Microsoft Edge suporta os seguintes tipos de SSO (single sign-on):
- SSO baseado em PRT
- SSO sem com com interações
- SSO baseado em WIA
