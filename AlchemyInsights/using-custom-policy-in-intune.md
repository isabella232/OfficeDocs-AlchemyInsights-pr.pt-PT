# <a name="using-custom-policy-in-intune"></a>Utilizar uma política personalizada no Intune

O Intune tem muitas definições incorporadas para dispositivos Android, iOS, macOS e Windows Android. Também pode criar um perfil personalizado e, em seguida, implementar estas definições personalizadas nos seus dispositivos com o Intune. Os perfis personalizados foram concebidos para adicionar definições e funcionalidades de dispositivos que não estão incorporadas no Intune.

[Definições personalizadas para dispositivos Android no Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Definições personalizadas para dispositivos Android Enterprise no Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Definições personalizadas para dispositivos iOS no Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Definições personalizadas para dispositivos macOS no Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Definições personalizadas para Windows 10 dispositivos no Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Definições personalizadas para Windows Holographic for Business dispositivos no Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
